**NetPrime**

A free movie and web series streaming application, because everyone deserves a good movie night.



**Tech Used**
**React Native** - Cross-platform mobile development framework

**Expo** - Development platform and tools for React Native

**TypeScript** - Type-safe JavaScript

**React Navigation** - Navigate between screens and manage navigation stack

**Axios** - HTTP client for API requests

**Expo Router** - File-based routing for React Native

**React Native Reanimated** - Smooth animations and gestures

**React Native Web View** - Run web on the app
